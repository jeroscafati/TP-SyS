Sobre el Proyecto

Este proyecto fue desarrollado como parte de la materia Señales y Sistemas de la Universidad Nacional de Tres de Febrero, con el objetivo de crear una herramienta práctica para el análisis de parámetros acústicos según la norma ISO 3382.

Objetivos Principales

  • Implementar algoritmos para el cálculo de parámetros acústicos
  • Crear una interfaz intuitiva para el análisis de datos
  • Facilitar el estudio de la acústica de recintos

Características Técnicas

  • Desarrollado en Python con Flask
  • Código abierto y disponible en GitHub
  • Visualización interactiva de datos
